# fast-memoize

## Benchmark

This lib tries to be the fast memoization possible that supports `N` arguments.
Below you can see a benchmark against some popular memoization libraries.

<img src="http://rawgit.com/caiogondim/fast-memoize/master/img/benchmark.png" width="100%" />

The reason *fast-memoize* is not faster than *lodash* is because lodash only
support one argument, while fast-memoize supports *N* arguments.
